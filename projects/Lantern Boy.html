Lantern Boy

This game is an entry to the 2023 GDKO game jam. The task was to create a level with a memorable location.
Play the game on itch.io


For the 2023 GDKO I created an entry for the theme ‘Memorable location’. I have imagined a small riverside ghost town, where the player must guide ghosts to become the colours they've always wanted to be!


In two weeks, I created all the assets from the art, code and soundtrack. I took this opportunity to create a completed project and refine my art and production skills.
During this project, I wanted to experiment with 2D technical art, particularly in Godot which I was unfamiliar with at the time. I took the time to understand simple concepts such as 2D lighting (which serves as the main mechanic of the game), as well as parallax backgrounds and layers. For the water, I implemented a reflective shader in Godot’s shading language.
